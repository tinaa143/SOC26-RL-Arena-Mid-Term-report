# Week 1 Grid Simulation

This project is a simple grid-based simulation where a player moves inside a 3 x 10 board.

# Symbols

"P" = Player
 "X" = Obstacle
"." = Empty cell

# Controls

Use the following keys to move the player:

"W" = Up
"A" = Left
"S" = Down
"D" = Right


# Rules

The player cannot move through obstacles.
If a move is invalid, the program prints "Invalid Move".
 Obstacles are placed on the edges of the grid.

#Run
 
"python main.py" run this terminal
